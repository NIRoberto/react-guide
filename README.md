# React × Airbnb — Full Course

A complete React curriculum built around a real Airbnb-style app. Each phase builds on the previous one — by the end you have a production-ready, fully typed, tested, and deployed application.

---

## Course Structure

| Phase | Topics | Level |
|-------|--------|-------|
| [Phase 1 — Foundations](./docs/phase-1.md) | JSX, Components, Props, useState, Events | Beginner |
| [Phase 2 — Hooks & Styling](./docs/phase-2.md) | useEffect, useRef, useContext, useReducer, CSS Modules, Tailwind | Intermediate |
| [Phase 3 — Advanced](./docs/phase-3.md) | React Router v6, Zustand, React.memo, Virtualization | Advanced |
| [Phase 4 — Real-World Skills](./docs/phase-4.md) | TanStack Query, Multi-Step Forms, Validation, Testing | Advanced |
| [Phase 5 — Patterns & TypeScript](./docs/phase-5.md) | HOCs, Compound Components, Generics, Full TypeScript | Expert |
| [Phase 6 — Production](./docs/phase-6.md) | Vite Build, Vercel, CI/CD, Next.js App Router | Expert |

---

## What You Build

A full Airbnb clone that evolves across all 6 phases:

```
Phase 1 → Static listing cards with props and .map()
Phase 2 → Live search, favorites via Context, CSS Modules styling
Phase 3 → Multi-page app with routing, global Zustand store
Phase 4 → Cached data fetching, booking form, test suite
Phase 5 → Compound components, HOCs, full TypeScript types
Phase 6 → Production build, Vercel deploy, Next.js migration
```

---

## Getting Started

```bash
npm install
npm run dev
```
